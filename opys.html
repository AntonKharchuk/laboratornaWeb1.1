Опис предметної галузі

Питання "Який фільм обрати?" бентежить багатьох щоразу коли настає час подивитися кіно. Я пропоную сервіс, що буде підбирати фільм за примхами користувача. Ця розробка особливо актуальна напередодні довгих зимових вечорів.

Мета: написання сайту з каталогом фільмів для перегляду. Довавання до сайту відповідних елементів для зручного пересування між сторінками. Актуалізація данних для зручності користувача.

Об'єктом дослідження є: пошукова система заточена на фільми.

Предметом дослідження є: аналіз роботи існуючих пошукових систем.

Постановка задачі: розробка веб-додатку, з допомогою якого користувач знаходить інформацію про популярні фільми, отримує посилання на фільм для перегляду. Розробка алгоритмів сортування та підбору фільмів. Додавання гнучкої системи данних для постійного їх оновлення.
